# Jeeya Chollangi — Portfolio Website

A modern, premium dark-theme developer portfolio built with HTML5, CSS3, and JavaScript.

🔗 **Live Site**: [jeeyachollangi.github.io/portfolio)

## Tech Stack
- HTML5 (Semantic)
- Vanilla CSS3 (Glassmorphism, Animations)
- JavaScript (ES6+)
- FontAwesome Icons
- Google Fonts (Outfit, Inter, Fira Code)

## Features
- Full-page section scrolling
- Particle background animation
- Custom cursor effects
- Scroll-triggered animations
- Skill progress bars
- Responsive (mobile, tablet, desktop)
- Contact form
- Download Resume button
- Certificate viewer
